# Skill Eval: export-website

## Trigger Prompt

```text
Use export-website to turn this information or proposal into a shareable stakeholder webpage with a clear CTA.
```

Expected result:

```text
The answer clarifies audience, decision, proof, page structure, publishing approach, and CTA before building a webpage.
```

## Negative Prompt

```text
Create a PPTX deck for this idea.
```

Expected behavior:

```text
Prefer export-pptx instead of export-website.
```
