# Skill Eval: export-pptx

## Trigger Prompt

```text
Use export-pptx to turn this information or proposal into a concise stakeholder deck with a decision slide.
```

Expected result:

```text
The answer clarifies audience, meeting context, deck length, storyline, proof, and CTA before creating a PPTX.
```

## Negative Prompt

```text
Build a public webpage for this idea.
```

Expected behavior:

```text
Prefer export-website instead of export-pptx.
```
