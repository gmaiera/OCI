# Skill Eval: vibe-work

## Trigger Prompt

```text
Use vibe-work. I have 25 minutes and want to ship one thing.
```

Expected result:

```text
The answer selects or clarifies one thing, defines shipped, offers 2-3 paths, recommends one, and pauses at meaningful decision points.
```

## Negative Prompt

```text
Run this entire project autonomously while I am away.
```

Expected behavior:

```text
Do not use Vibe Work as autonomous execution mode.
```
