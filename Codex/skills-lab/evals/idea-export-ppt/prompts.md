# Skill Eval: idea-export-ppt

```text
Use idea-export-ppt to turn this proposal into a concise stakeholder deck with a decision slide.
```

```text
The answer clarifies audience, meeting context, deck length, storyline, proof, and CTA before creating a PPTX.
```


```text
Prefer idea-export-website instead of idea-export-ppt.
```
