# Skill Eval: idea-export-website

```text
Use idea-export-website to turn this proposal into a shareable stakeholder webpage with a clear CTA.
```

Expected result:

```text
The answer clarifies audience, decision, proof, page structure, publishing approach, and CTA before building a webpage.
```


Expected behavior:

```text
Prefer idea-export-ppt instead of idea-export-website.
```
